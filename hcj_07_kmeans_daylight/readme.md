
## Ham Cam : K-Means Daylight

[K-means](https://en.wikipedia.org/wiki/K-means_clustering) is a method for clustering input data.

For this example, the [armadillo](https://arma.sourceforge.net) k-means function is used to cluster average RGB's per frame blueness into k=2 clusters.

Previously [anomaly detection](hcc_06_blueness_anomalies/) was applied to the average blueness. In this case the average RGB's are used directly.

For k-means with two clusters, presumably the clustering results should partition the frames into daylight (and night-time).
